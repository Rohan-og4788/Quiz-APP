# 🧠 Quiz App

A responsive and interactive Quiz Application built using modern web technologies.  
This project allows users to test their knowledge through multiple-choice questions with a clean UI and smooth user experience.


## 🚀 Features

- 📋 Multiple-choice quiz questions
- ✅ Instant answer checking
- 📊 Score calculation system
- 🔄 Next question navigation
- 🎨 Responsive and attractive UI
- ⚡ Fast and lightweight performance
- 📱 Mobile-friendly design


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
